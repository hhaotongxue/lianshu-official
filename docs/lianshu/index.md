# 炼书

炼书是一个本地AI Agent + 笔记助手，让AI成为你的知识伙伴。

## 核心特性

### 🤖 本地AI Agent

- 完全在本地运行，保护你的隐私
- 支持多种AI模型
- 智能对话和问答
- 上下文理解和记忆

### 📝 智能笔记助手

- AI辅助写作和编辑
- 自动摘要和总结
- 智能标签和分类
- 知识图谱构建

### 🔍 强大的搜索

- 语义搜索
- 全文检索
- 标签过滤
- 时间线浏览

### 🔗 无缝集成

- 与炼书编辑器深度集成
- 支持多种笔记格式
- 导入导出功能
- API扩展支持
