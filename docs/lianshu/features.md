# 功能特性

## AI能力

### 智能对话

炼书内置强大的AI对话能力，可以：

- 回答问题
- 提供建议
- 辅助写作
- 代码生成

### 上下文理解

AI能够理解你的笔记上下文：

- 记住之前的对话
- 关联相关笔记
- 提供个性化建议

### 多模型支持

支持多种AI模型：

- OpenAI GPT系列
- Anthropic Claude
- 本地模型（Llama、Mistral等）
- 自定义模型

## 笔记管理

### 智能组织

- 自动标签
- 智能分类
- 关系图谱
- 时间线视图

### 强大搜索

- 全文搜索
- 语义搜索
- 标签过滤
- 日期范围

### 导入导出

支持多种格式：

- Markdown
- HTML
- PDF
- JSON

## 扩展性

### 插件系统

- 丰富的插件生态
- 自定义插件开发
- 插件市场

### API接口

- RESTful API
- WebSocket支持
- Webhook集成

### 主题定制

- 多种预设主题
- 自定义CSS
- 暗色模式
