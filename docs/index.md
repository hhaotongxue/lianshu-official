## 为什么选择炼书？

炼书是一个完整的笔记解决方案，结合了AI助手和专业编辑器的优势：

- **炼书（Lianshu）**：本地AI Agent + 笔记助手，让AI成为你的知识伙伴
- **炼书编辑器（Lianshu Editor）**：专业的Markdown编辑器，专注于写作体验

## 快速开始

```bash
# 安装炼书
bun install lianshu

# 或者使用炼书编辑器
bun install lianshu-editor
```
