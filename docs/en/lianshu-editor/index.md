# Lianshu Editor

Lianshu Editor is a Markdown editor designed specifically for note-taking, providing a smooth writing experience and rich feature extensions.

## Core Features

### ✍️ Smooth Editing Experience

- WYSIWYG Markdown editing
- Real-time preview
- Syntax highlighting
- Keyboard shortcut support

### 🎨 Rich Format Support

- Standard Markdown syntax
- Extended syntax (tables, task lists, footnotes, etc.)
- Code block highlighting
- Math formulas (LaTeX)
- Diagrams (Mermaid)

### 📁 File Management

- File tree browsing
- Quick search
- Tag management
- Favorites

### 🔌 Extensibility

- Plugin system
- Custom themes
- Keyboard shortcut configuration
- API interface

### 🌙 Themes

- Light/dark theme toggle
- Custom color schemes
- Multiple preset themes

## Integration with Lianshu

Lianshu Editor is deeply integrated with the Lianshu AI assistant, providing:

- AI-assisted writing
- Smart completion
- Auto summarization
- Content optimization suggestions
