# Features

## Editing Experience

### WYSIWYG

- Real-time preview
- Instant rendering
- Smooth input
- Zero-latency response

### Syntax Support

#### Basic Markdown

```markdown
# Heading
**Bold** *Italic*
- List
[Link](url)
![Image](url)
```

#### Extended Syntax

- Tables
- Task lists
- Footnotes
- Highlights
- Superscript/Subscript

#### Code Highlighting

Supports syntax highlighting for 100+ programming languages:

```javascript
function hello() {
  console.log('Hello, World!')
}
```

#### Math Formulas

Supports LaTeX math formulas:

```latex
$$
E = mc^2
$$
```

#### Diagrams

Supports Mermaid diagrams:

```mermaid
graph TD
  A[Start] --> B[Edit]
  B --> C[Save]
```
