# Features

## AI Capabilities

### Intelligent Conversation

Lianshu has powerful AI conversation capabilities:

- Answer questions
- Provide suggestions
- Writing assistance
- Code generation

### Context Understanding

AI understands your note context:

- Remember previous conversations
- Link related notes
- Provide personalized suggestions

### Multi-Model Support

Supports various AI models:

- OpenAI GPT series
- Anthropic Claude
- Local models (Llama, Mistral, etc.)
- Custom models

## Note Management

### Smart Organization

- Auto-tagging
- Intelligent categorization
- Relationship graph
- Timeline view

### Powerful Search

- Full-text search
- Semantic search
- Tag filtering
- Date range

### Import/Export

Supports multiple formats:

- Markdown
- HTML
- PDF
- JSON

## Extensibility

### Plugin System

- Rich plugin ecosystem
- Custom plugin development
- Plugin marketplace

### API Interface

- RESTful API
- WebSocket support
- Webhook integration

### Theme Customization

- Multiple preset themes
- Custom CSS
- Dark mode
