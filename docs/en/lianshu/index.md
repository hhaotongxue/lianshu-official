# Lianshu

Lianshu is a local AI Agent + note assistant that makes AI your knowledge companion.

## Core Features

### 🤖 Local AI Agent

- Runs completely locally, protecting your privacy
- Supports multiple AI models
- Intelligent conversation and Q&A
- Context understanding and memory

### 📝 Smart Note Assistant

- AI-assisted writing and editing
- Automatic summarization
- Intelligent tagging and categorization
- Knowledge graph construction

### 🔍 Powerful Search

- Semantic search
- Full-text search
- Tag filtering
- Timeline browsing

### 🔗 Seamless Integration

- Deep integration with Lianshu Editor
- Supports multiple note formats
- Import/export functionality
- API extension support
