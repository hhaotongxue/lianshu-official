# 功能特性

## 编辑体验

### 所见即所得

- 实时预览
- 即时渲染
- 流畅输入
- 无延迟响应

### 语法支持

#### 基础Markdown

```markdown
# 标题
**粗体** *斜体*
- 列表
[链接](url)
![图片](url)
```

#### 扩展语法

- 表格
- 任务列表
- 脚注
- 高亮
- 上下标

#### 代码高亮

支持100+编程语言的语法高亮：

```javascript
function hello() {
  console.log('Hello, World!')
}
```

#### 数学公式

支持LaTeX数学公式：

```latex
$$
E = mc^2
$$
```

#### 图表

支持Mermaid图表：

```mermaid
graph TD
  A[开始] --> B[编辑]
  B --> C[保存]
```
