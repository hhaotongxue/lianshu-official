# 炼书编辑器

炼书编辑器是一个专为笔记设计的Markdown编辑器，提供流畅的书写体验和丰富的功能扩展。

## 核心特性

### ✍️ 流畅的编辑体验

- 所见即所得的Markdown编辑
- 实时预览
- 语法高亮
- 快捷键支持

### 🎨 丰富的格式支持

- 标准Markdown语法
- 扩展语法（表格、任务列表、脚注等）
- 代码块高亮
- 数学公式（LaTeX）
- 图表（Mermaid）

### 📁 文件管理

- 文件树浏览
- 快速搜索
- 标签管理
- 收藏夹

### 🔌 扩展性

- 插件系统
- 自定义主题
- 快捷键配置
- API接口

### 🌙 主题

- 明暗主题切换
- 自定义配色
- 多种预设主题

## 与炼书集成

炼书编辑器与炼书AI助手深度集成，提供：

- AI辅助写作
- 智能补全
- 自动摘要
- 内容优化建议
